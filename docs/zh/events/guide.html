# 说明

提示

eventsHelper是帮助做事件监听的工具，也可以通过VxeGridWrap组件监听vxe-grid的事件。

# 基本使用

# 引入

import { eventsHelper } from 'vxe-table-middleware';
1

# 使用

  1. 调用eventsHelper方法，返回配置实例，通过实例的on方法注册监听事件
  2. 最后将实例对象传递给 grid 构建工具useVxeGrid

# eventsHelper 实例方法

# function on(event, callback)

注册事件监听

Parameters

  • event (string) - 事件名称
  • callback (function) - 回调函数

# function off(event)

移除指定事件监听

Parameters

  • event (string) - 事件名称
